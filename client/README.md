# Client

## Requirements

* NodeJS 10.X+

## Setup/Development

Installed dependencies:

```
npm i
```

Add an `.env` file. For example:

```
API_URL=http://127.0.0.1:5000
```

Start the development server:

```bash
npm run dev

# or start the server and open the app in a new browser tab
npm run dev -- --open
```
